1. How does JavaScript work??

A JavaScript engine used for understanding and running JavaScript files. Every browser have own JavaScript engine such as chrome has V8 engine , firefox has Spidermonkey , safari has JavaScript Core ,Edge has chakra. Every time the JavaScript engine 'parser' the JavaScript file and checks JavaScript code for error in line by line .If there is no error then make a data structure called Abstract Syntax Tree. Then the JavaScript engine converts it to the machine code.This converted code sends the system for execution.

2. What is the difference between local storage and session storage ??

Main difference between local storage and session storage is the different expiration time. local storage is permanent but session storage temporary.session storage will active or accessible while the window that created it is open.If the window once is closed, the stored data is deleted.local storage data keep and accessible until the user delete it.There is another different between local storage and session storage.Local storage used for long term use and session storage used for temporary use.
